# 思路
- 思路 1：
按照题目说明模拟即可，中途记录步骤数
注意题目给的字符串长度为 $<500$，不能转换成整数再模拟，只能用字符串模拟


```cpp
class Solution {
public:
    int numSteps(string s) {
        int cnt = 0;
        while(s != "1"){
            if(s.back() == '1'){
                int r = 1;
                for(int i = s.length() - 1; i >= 0; i--){
                    if(r == 1){
                        if(s[i] == '0'){
                            s[i] = '1';
                            r = 0;
                        }
                        else{
                            s[i] = '0';
                        }
                    }
                }
                if(r) s = "1" + s;
            }
            else{
                s.pop_back();
            }
            cnt++;
        }
        return cnt;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
