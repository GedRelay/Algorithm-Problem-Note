# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int f(int n)
{
	if (n == 1)return 1;
	return 2 * (f(n - 1) + 1);
}
int main()
{
	int n;
	cin >> n;
	cout << f(n);
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
