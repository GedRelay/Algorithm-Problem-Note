# 思路
- 思路 1：
入栈：把队列看作一个环，每次入栈后都将队头出队并加入队尾直到队列满足元素的逆序存放
其他操作照旧


```cpp
class MyStack {
public:
    queue<int>q;
    MyStack() {

    }
    
    void push(int x) {
        int n = q.size();
        q.push(x);
        while(n--){
            int x = q.front();
            q.pop();
            q.push(x);
        }
    }
    
    int pop() {
        int x = q.front();
        q.pop();
        return x;
    }
    
    int top() {
        return q.front();
    }
    
    bool empty() {
        return q.empty();
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
