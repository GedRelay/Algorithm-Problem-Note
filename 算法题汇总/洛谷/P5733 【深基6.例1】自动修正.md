# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	char s[110];
	cin >> s;
	for (int i = 0; s[i] != '\0'; i++)
		if ('a' <= s[i] && s[i] <= 'z')
			s[i] = s[i] - 'a' + 'A';
	cout << s;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
