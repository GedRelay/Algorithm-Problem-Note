# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int a[100000001];
int main()
{
	int n, sum = 0;
	cin >> n;
	for (int i = 0; i < n; i++)
	{
		cin >> a[i];
		sum += a[i] % 2;
	}
	if (sum % 2)cout << "Alice";
	else cout << "Bob";
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
