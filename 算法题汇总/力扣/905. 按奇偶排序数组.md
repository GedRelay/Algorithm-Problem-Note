# 思路
- 思路 1：
双指针算法
指针 `i` 遍历数组，指针 `j` 始终指向已确定的偶数的后面
若指针 `i` 遇到了偶数，则将其与 `j` 位置的数交换，然后 `j++` 

```cpp
class Solution {
public:
    vector<int> sortArrayByParity(vector<int>& nums) {
        // j始终指向已确定的偶数的后面
        for(int i = 0, j = 0; i < nums.size(); i++){
            if(nums[i] % 2 == 0){
                swap(nums[i], nums[j]);
                j++;
            }
        }
        return nums;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
