# 思路
- 思路 1：
对原数组进行排序，之后遍历，只需要满足每一个元素都比前面的一个元素大即可


```cpp
class Solution {
public:
    int minIncrementForUnique(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        int res = 0;
        for(int i = 1; i < nums.size(); i++){
            if(nums[i] <= nums[i - 1]){
                res += nums[i - 1] + 1 - nums[i];
                nums[i] = nums[i - 1] + 1;
            }
        }
        return res;
    }
};
```


- 时间复杂度：${O\left( n\log n \right)  }$ 
- 空间复杂度：${O\left( \log n \right)  }$
