- 思路 1：

```cpp
#include<iostream>
using namespace std;
int main()
{
	int hand, apple[10], ans = 0;
	for (int i = 0; i < 10; i++)
		cin >> apple[i];
	cin >> hand;
	for (int i = 0; i < 10; i++)
		if (30 + hand >= apple[i])ans++;
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
