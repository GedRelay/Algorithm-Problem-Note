# 思路
- 思路 1：
二分答案
函数：f (锯子的高度) = 得到的木材长度
趋势：递减
条件：f (x) >= M
目标：找到最后一个高度 x
使用模型 2


```cpp
#include<iostream>
#include<algorithm>
using namespace std;
typedef long long LL;
int n, m;
int T[1000100];

LL f(int x){
    // 锯子高度为x时，切割的木头总长度
    LL res = 0;
    for(int i = 0; i < n; i++){
        if(T[i] > x) res += T[i] - x;
    }
    return res;
}

int main(){
    cin >> n >> m;
    for(int i = 0; i < n; i++){
        cin >> T[i];
    }
    int l = 0, r = 4e5;
    while(l < r){
        int mid = l + r + 1 >> 1;
        if(f(mid) >= m) l = mid;
        else r = mid - 1;
    }
    cout << l << endl;
    return 0;
}
```


- 时间复杂度：${O\left( n\log r \right)  }$，其中 ${r }$ 为答案范围
- 空间复杂度：${O\left( n \right)  }$
