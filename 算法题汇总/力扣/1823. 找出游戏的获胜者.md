```cpp
class Solution {
public:
    int findTheWinner(int n, int k) {
        int index = 0;
        for(int i = 2; i <= n; i++){
            index = (index + k) % i;
        }
        return index + 1;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
