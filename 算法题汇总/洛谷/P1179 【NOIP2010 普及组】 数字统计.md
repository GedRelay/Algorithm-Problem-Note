# 思路
- 思路 1：
枚举 $[L, R]$ 中的所有数，然后统计每个数中 $2$ 的个数 


```cpp
#include<iostream>
using namespace std;
int main()
{
	long long M, N, p, k = 0;
	cin >> M >> N;//依次得到[M,N]内的数
	for (int n = M; n <= N; n++) /*获得从M到N的整数n*/
	{
		int y = n;//换一个字母不然会造成无限循环
		while (y>0) //直到无法除以10换下一个数
		{
			p = y % 10;//得到个位数
			if (p == 2) { k++;}//出现2则k+1
			y = y / 10;//去掉个位数
		}
	}
	cout << k;//输出个数
	return 0;
}
```


- 时间复杂度：${O\left( \left( R-L \right) \log R \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
