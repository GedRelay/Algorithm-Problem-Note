# 思路
- 思路 1：
由题意，最小的那个数的范围一定在 ${123\sim 333 }$ 之间，只需要枚举之间的所有数再判断即可


```cpp
#include<iostream>
using namespace std;

int main()
{
	for (int i = 123; i <= 333; i++){
	    int a = i, b = i * 2, c = i * 3;
		int m[10] = { 0 }; // 记录每个数字出现多少遍
		while(a){
		    m[a % 10]++;
		    a /= 10;
		}
		while(b){
		    m[b % 10]++;
		    b /= 10;
		}
		while(c){
		    m[c % 10]++;
		    c /= 10;
		}
		int cnt = 0;
		for(int j = 1; j <= 9; j++){
		    if(m[j] == 1) cnt++;
		}
		if(cnt == 9){ // 每个数字都出现一遍
		    cout << i << " " << 2 * i << " " << 3 * i << endl;
		}
	}
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
