# 思路
- 思路 1：
贪心
某一天桌子上有 $x$，则第二天会有 $x-1$，再下一天会有 $x-2$，。。。直到 $2$ 


```cpp
class Solution {
public:
    int distinctIntegers(int n) {
        return n == 1 ? 1 : n - 1;
    }
};
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
