# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;

int n;
int a[110];

int main()
{
    cin >> n;
    int sum = 0;
    for(int i = 1; i <= n; i++){
        cin >> a[i];
        sum += a[i];
    }
    int avg = sum / n;
    int cnt = 0;
    for(int i = 1; i <= n; i++){
        if(a[i] != avg){
            a[i + 1] -= avg - a[i];
            cnt++;
        }
    }
    cout << cnt;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
