# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, a[25][25];
	cin >> n;
	for (int i = 1; i <= n; i++)
	{
		for (int j = 1; j <= i; j++)
		{
			if (j == 1 || i == j)
				a[i][j] = 1;
			else
				a[i][j] = a[i - 1][j] + a[i - 1][j - 1];
		}
	}
	for (int i = 1; i <= n; i++)
		for (int j = 1; j <= i; j++)
			cout << a[i][j] << (j == i ? '\n' : ' ');
	return 0;
}
```


- 时间复杂度：${O\left( n^{2}  \right)  }$ 
- 空间复杂度：${O\left( n^{2}  \right)  }$
