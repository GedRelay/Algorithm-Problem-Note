# 思路
- 思路 1：


```cpp
#include<cmath>
#include<iostream>
using namespace std;
int main()
{
	int n;
	cin >> n;
	int m = (sqrt(8.0 * n + 1) - 1) / 2;
	int ans;
	ans = m * (m + 1) * (2 * m + 1) / 6 + (m + 1) * (n - ((1 + m) * m / 2));
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( k \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
