# 思路
- 思路 1：
首先统计 `"VK"` 的出现次数，再统计时将其转换为其他字符，如 `"aa"` 
之后再次遍历字符串，查找有没有出现 `"VV"` 或者 `"KK"`，如果有则答案 `+1` 


```cpp
#include<cstring>
#include<iostream>
using namespace std;
int main()
{
	int n, ans = 0;
	string s;
	cin >> n >> s;
	for (int i = 1; i < n; i++)
	{
		if (s[i] == 'K' && s[i - 1] == 'V')
		{
			s[i - 1] = 'a';
			s[i] = 'a';
			ans++;
		}
	}
	for (int i = 1; i < n; i++)
	{
		if ((s[i] == 'V' && s[i - 1] == 'V')||(s[i] == 'K' && s[i - 1] == 'K'))
		{
			ans++;
			break;
		}
	}
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
