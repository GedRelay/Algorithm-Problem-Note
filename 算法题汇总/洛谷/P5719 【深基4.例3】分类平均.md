# 思路
- 思路 1：


```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
	int Asum = 0, Bsum = 0, n, k, A = 0, B = 0;
	scanf("%d%d",&n,&k);
	for (int i = 1; i <= n; i++)
	{
		if (i % k) 
		{
			Bsum += i; 
			B++;
		}
		else
		{
			Asum += i;
			A++;
		}
	}
	printf("%.1f %.1f", 1.0*Asum / A, 1.0*Bsum / B);
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
