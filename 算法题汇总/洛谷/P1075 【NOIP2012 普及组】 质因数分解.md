# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n;
	cin >> n;
	for (int i = 2; i < n; i++)
		if (n % i == 0){
			cout << n / i;
			break;
		}
	return 0;
}
```


- 时间复杂度：${O\left( \sqrt{ n }  \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
