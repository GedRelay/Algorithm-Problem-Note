# 思路
- 思路 1：


```cpp
#include<iostream>
#include<cstring>
using namespace std;
int main()
{
	string s;
	int ans = 0;
	while (cin >> s)
		ans += s.length();
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
