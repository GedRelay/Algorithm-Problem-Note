# 思路
- 思路 1：
任意两位同学之间的最大值取决于这两位同学最小距离的较大值，将较大值加和即可


```cpp
#include<iostream>
using namespace std;

int a[100010];

int main()
{
	int n;
	cin >> n;
	for(int i = 0; i < n; i++) cin >> a[i];
	long long ans = 0;
	for(int i = 1; i < n; i++){
	    ans += max(a[i - 1], a[i]);
	}
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
