# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int ans[13] = { 0,31,28,31,30,31,30,31,31,30,31,30,31 };
	int year, month;
	cin >> year >> month;
	if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0))
	{
		ans[2] = 29;
	}
	cout << ans[month];
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
