# 思路
- 思路 1：


```cpp
#include<bits/stdc++.h>
using namespace std;
int n, m, a[10010];
int main()
{
	cin >> n >> m;
	for (int i = 1; i <= n; i++)
		scanf("%d", &a[i]);
	while (m--)
		next_permutation(a + 1, a + n + 1);
	for (int i = 1; i <= n; i++)
		printf("%d ", a[i]);
	return 0;
}
```


- 时间复杂度：${O\left( nm \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
