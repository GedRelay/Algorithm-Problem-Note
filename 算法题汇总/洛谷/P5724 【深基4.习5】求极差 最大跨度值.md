# 思路
- 思路 1：
使用擂台法求最大和最小值


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, x, max = 0, min = 1000;
	cin >> n;
	for (int i = 0; i < n; i++)
	{
		cin >> x;
		max = x > max ? x : max;
		min = x < min ? x : min;
	}
	cout << max - min;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
