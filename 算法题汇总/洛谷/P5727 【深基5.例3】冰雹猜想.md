# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, a[220], cnt = 0;
	cin >> n;
	while (n != 1)
	{
		a[++cnt] = n;
		if (n % 2)n = n * 3 + 1;
		else n /= 2;
	}
	cout << "1 ";
	for (int i = cnt; i >= 1; i--)
		cout << a[i] << " ";
	return 0;
}
```


- 时间复杂度：${O\left( 未解之谜 \right)  }$ 
- 空间复杂度：${O\left( 未解之谜 \right)  }$
