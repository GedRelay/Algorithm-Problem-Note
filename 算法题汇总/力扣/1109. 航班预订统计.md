# 思路
- 思路 1：

模板题


```cpp
class Solution {
public:
    vector<int> corpFlightBookings(vector<vector<int>>& bookings, int n) {
        vector<int> d(n, 0);
        for(auto book : bookings){
            int first = book[0], last = book[1], seats = book[2];
            d[first - 1] += seats;
            if(last != n) d[last] -= seats;
        }
        for(int i = 1; i < n; i++){
            d[i] += d[i - 1];
        }
        return d;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
