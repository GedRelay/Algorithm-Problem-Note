# 思路
- 思路 1：
枚举每个数，对每个数进行统计


```cpp
#include<iostream>
using namespace std;
int main()
{
	int cnt[10] = { 0 };
	int a, b;
	cin >> a >> b;
	for (int i = a; i <= b; i++)
	{
		int t = i;
		while (t)
		{
			cnt[t % 10]++;
			t /= 10;
		}
	}
	for (int i = 0; i < 10; i++)
		cout << cnt[i] << " ";
	return 0;
}
```


- 时间复杂度：${O\left( \left( N-M \right) \log N \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
