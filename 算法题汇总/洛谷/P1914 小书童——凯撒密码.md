# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n;
	cin >> n;//输入n
	string password;
	cin >> password;//输入解码前的密码

	for (int i = 0; i < password.length(); i++)//每一个字母单独操作
	{

		for (int j = 0; j < n; j++)//循环加1，加n次
		{
			password[i]++;
			if (password[i] == ('z' + 1))
				password[i] = 'a';
		}
		cout << password[i];
	}
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
