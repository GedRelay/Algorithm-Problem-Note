# 思路
- 思路 1：
模拟
使用数组来表示某个位置是否种数，`0` 代表有树，`1` 代表没有树


```cpp
#include<iostream>
using namespace std;

int l, m;
bool road[10010] = {0}; // 0代表有树，1代表没有树

int main(){
    cin >> l >> m;
    while(m--){
        int a, b;
        cin >> a >> b;
        for(int i = a; i <= b; i++){
            road[i] = 1; // 拔树
        }
    }
    // 统计还剩多少树
    int cnt = 0;
    for(int i = 0; i <= l; i++){
        cnt += 1 - road[i];
    }
    cout << cnt;
    return 0;
}
```


- 时间复杂度：${O\left( ml \right)  }$ 
- 空间复杂度：${O\left( l \right)  }$
