# 思路
- 思路 1：
枚举所有学生对判断即可


```cpp
#include<cmath>
#include<iostream>
using namespace std;
int main()
{
	int n, a[1010][3], ans = 0;
	cin >> n;
	for (int i = 1; i <= n; i++)
		cin >> a[i][0] >> a[i][1] >> a[i][2];
	for (int i = 1; i < n; i++)
		for (int j = i + 1; j <= n; j++)
			if (abs(a[i][0] - a[j][0]) <= 5
				&& abs(a[i][1] - a[j][1]) <= 5
				&& abs(a[i][2] - a[j][2]) <= 5
				&& abs(a[i][0] + a[i][1] + a[i][2] - a[j][0] - a[j][1] - a[j][2]) <= 10)
				ans++;
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( n^{2}  \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
