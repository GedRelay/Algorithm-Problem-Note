# 思路
- 思路 1：

模板题


```cpp
//20201104
#include<iostream>
using namespace std;
const int fac[] = { 1,1,2,6,24,120,720,5040,40320,362880 };
int main()
{
	int n, x, a[12], ans = 0;
	cin >> n >> x;
	for (int i = n; i >= 1; i--)
	{
		a[i] = x % 10;
		x /= 10;
	}
	for (int i = 1; i <= n-1; i++)
	{
		int b = 0;
		for (int j = i + 1; j <= n; j++)
			if (a[i] > a[j])b++;
		ans += b * fac[n - i];
	}
	cout << ans + 1;
	return 0;
}
```


- 时间复杂度：${O\left( n^{2}  \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
