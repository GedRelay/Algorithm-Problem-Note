# 思路
- 思路 1：
两边收缩的双指针

```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        for(int l = 0, r = s.size() - 1; l < r; l++, r--){
            swap(s[l], s[r]);
        }
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
