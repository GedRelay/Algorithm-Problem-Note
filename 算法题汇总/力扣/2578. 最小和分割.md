# 思路
- 思路 1：
贪心
将数字按位进行排序，然后依次均匀分配到两个数中，这样的和是最小的


```cpp
class Solution {
public:
    int splitNum(int num) {
        vector<int> nums;
        while(num){
            nums.push_back(num % 10);
            num /= 10;
        }
        sort(nums.begin(), nums.end());
        int num1 = 0, num2 = 0;
        for(int i = 0; i < nums.size(); i += 2){
            num1 = num1 * 10 + nums[i];
            if(i + 1 < nums.size())
                num2 = num2 * 10 + nums[i + 1];
        }
        return num1 + num2;
    }
};
```


- 时间复杂度：${O\left( n\log n \right)  }$, ${n }$ 为数字长度
- 空间复杂度：${O\left( n \right)  }$
