# 思路
- 思路 1：
将其包含 $3$ 的因子全部除掉，最后判断剩下的数是不是 $1$ 

```cpp
class Solution {
public:
    bool isPowerOfThree(int n) {
        while(n && n % 3 == 0){
            n /= 3;
        }
        return n == 1;
    }
};
```


- 时间复杂度：${O\left( \log n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
