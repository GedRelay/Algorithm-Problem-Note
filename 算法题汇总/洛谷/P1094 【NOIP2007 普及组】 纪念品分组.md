# 思路
- 思路 1：


```cpp
#include<bits/stdc++.h>
using namespace std;
int a[30010], w, n, cnt;
int main()
{
	cin >> w >> n;
	for (int i = 1; i <= n; i++)
		scanf("%d", &a[i]);
	sort(a + 1, a + n + 1);
	for (int l = 1, r = n; l <= r; r--,cnt++){
		if (a[l] + a[r] <= w)
			l++;
	}
	cout << cnt;
	return 0;
}
```


- 时间复杂度：${O\left( n\log n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
