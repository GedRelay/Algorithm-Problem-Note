# 思路
- 思路 1：


```cpp
#include<algorithm>//sort()
#include<iostream>
using namespace std;
int main()
{
	int a[3];
	char p[3];
	cin >> a[0] >> a[1] >> a[2];
	sort(a, a + 3);//升序排序
	cin >> p[0] >> p[1] >> p[2];
	cout << a[p[0] - 'A'] << " " << a[p[1] - 'A'] << " " << a[p[2] - 'A'];
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
