# 思路
- 思路 1：


```cpp
class Solution {
public:
    string addStrings(string num1, string num2) {
        string ans = "";
        int i = num1.length() - 1, j = num2.length() - 1;
        int r = 0;// 进位
        while(i >= 0 || j >= 0){
            if(i >= 0) r += num1[i--] - '0';
            if(j >= 0) r += num2[j--] - '0';
            ans += char(r % 10 + '0');
            r /= 10;
        }
        if(r) ans += char(r + '0');
        reverse(ans.begin(), ans.end());
        return ans;
    }
};
```


- 时间复杂度：${O\left( n + m\right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
