# 思路
- 思路 1：
模拟
按照顺序遍历数组，统计连续的 `0` 或 `1` 的数量即可


```cpp
#include <iostream>
using namespace std;
int main()
{
	string str, s;
	while (cin >> s)str += s;
	cout << s.length() << " ";
	char ch = '0';
	int sum = 0;
	for (int i = 0; i < str.length(); i++) {
		if (str[i] == ch)sum++;
		else {
			cout << sum << " ";
			sum = 1;
			ch = str[i];
		}
	}
	cout << sum;
	return 0;
}
```


- 时间复杂度：${O\left( n^{2}  \right)  }$ 
- 空间复杂度：${O\left( n^{2}  \right)  }$
