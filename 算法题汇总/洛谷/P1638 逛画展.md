# 思路
- 思路 1：
区间越长越容易满足要求，所有可以使用滑动窗口


```cpp
#include<iostream>
using namespace std;

int n, m;
int a[1000010];
int b[2001];

int main(){
    cin >> n >> m;
    for(int i = 1; i <= n; i++) cin >> a[i];
    
    int cnt = 0; // 记录有几个画家
    int minlen = 1000010, L = 0, R = 0;
    for(int l = 1, r = 1; r <= n; r++){
        if(b[a[r]]++ == 0) cnt++;
        while(l <= r && cnt == m){
            if(r - l + 1 < minlen){
                minlen = r - l + 1;
                L = l;
                R = r;
            }
            b[a[l]]--;
            if(b[a[l]] == 0) cnt--;
            l++;
        }
    }
    cout << L << " " << R;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
