# 思路
- 思路 1：


```cpp
class Solution {
public:
    vector<int> getConcatenation(vector<int>& nums) {
        int n = nums.size();
        vector<int> res(n * 2);
        for(int i = 0; i < n; i++){
            res[i] = nums[i];
            res[i + n] = nums[i];
        }
        return res;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
