# 思路
- 思路 1：


```cpp
class Solution {
public:
    int minDeletionSize(vector<string>& strs) {
        int n = strs.size(), m = strs[0].length();// 行数和列数
        int cnt = 0;
        for(int i = 0; i < m; i++){// 枚举每一列
            char pre = strs[0][i];
            for(int j = 1; j < n; j++){
                if(pre > strs[j][i]){
                    cnt ++;
                    break;
                }
                pre = strs[j][i];
            }
        }
        return cnt;
    }
};
```


- 时间复杂度：${O\left( nm \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
