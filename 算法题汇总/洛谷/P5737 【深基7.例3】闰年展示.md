# 思路
- 思路 1：


```cpp
//20201107
#include<iostream>
using namespace std;
bool runnian(int year)
{
	return ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0));
}
int main()
{
	int x, y, cnt = 0, rn[1500];
	cin >> x >> y;
	for (int i = x; i <= y; i++)
	{
		if (runnian(i))
			rn[++cnt] = i;
	}
	cout << cnt << endl;
	for (int i = 1; i <= cnt; i++)
		cout << rn[i] << " ";
	return 0;
}
```


- 时间复杂度：${O\left( y-x \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
