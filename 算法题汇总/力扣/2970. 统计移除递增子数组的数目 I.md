# 思路
- 思路 1：
暴力枚举两个端点

```cpp
class Solution {
public:
    int incremovableSubarrayCount(vector<int>& nums) {
        int n = nums.size();
        int ans = 0;
        // 枚举两个端点i和j
        for(int i = 0; i < n; i++){
            for(int j = i; j < n; j++){
                int t = -1;
                bool ok = true;
                for(int k = 0; k < n; k++){
                    if(k < i || k > j){
                        if(nums[k] <= t){
                            ok = false;
                            break;
                        }
                        t = nums[k];
                    }
                }
                if(ok) ans++;
            }
        }
        return ans;
    }
};
```


- 时间复杂度：${O\left( n^{3}  \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
