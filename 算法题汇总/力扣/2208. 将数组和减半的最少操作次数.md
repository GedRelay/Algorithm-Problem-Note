# 思路
- 思路 1：
贪心 + 堆
每次将最大值取出减半，直到和小于等于一半。动态获取最大值可以使用堆（优先队列）


```cpp
class Solution {
public:
    int halveArray(vector<int>& nums) {
        double sum = 0;
        priority_queue<double> q;
        for(int x : nums){
            sum += x;
            q.push(x);
        }
        int cnt = 0;
        double half = sum / 2;
        while(sum > half){
            double x = q.top();
            q.pop();
            sum -= x / 2;
            q.push(x / 2);
            cnt++;
        }
        return cnt;
    }
};
```


- 时间复杂度：${O\left( n\log n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
