# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int x, n, ans = 0;
	cin >> x >> n;//x是初始日期，n是天数
	for (int i = 0; i < n; i++)
	{
		if (x != 6 && x != 7) 
			ans += 250;
		x = x == 7 ? 1 : x + 1;
	}
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
