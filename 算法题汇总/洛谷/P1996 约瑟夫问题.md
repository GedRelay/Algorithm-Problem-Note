# 思路
- 思路 1：
使用队列进行模拟


```cpp
#include<queue>
#include<iostream>
using namespace std;
int main()
{
	queue <int> a;
	int n, k, temp;
	cin >> n >> k;
	for (int i = 1; i <= n; i++)
		a.push(i);//初始化
	while (!a.empty()){
		for (int i = 1; i < k; i++){
			temp = a.front();
			a.pop();
			a.push(temp);
		}
		cout << a.front() << " ";
		a.pop();
	}
	return 0;
}
```


- 时间复杂度：${O\left( n^{2}  \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
