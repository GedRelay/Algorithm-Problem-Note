# 思路
- 思路 1：
使用数组记录


```cpp
#include<iostream>
#include<vector>
using namespace std;
int main()
{
	vector <int> a;
	int n, m;
	cin >> n >> m;
	for (int i = 1; i <= n; i++)
	{
		int x;
		cin >> x;
		a.push_back(x);
	}
	while (m--)
	{
		int q;
		cin >> q;
		cout << a[q - 1] << endl;
	}
	return 0;
}
```


- 时间复杂度：${O\left( m \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
