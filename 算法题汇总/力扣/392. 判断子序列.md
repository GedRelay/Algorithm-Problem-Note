# 分类
#双指针/维护指针与探索指针 

# 思路
- 思路 1：
令指针 `i` 指向 `s`，指针 `j` 指向 `t` 
不断右移指针 `i` 直到找到 `s[i]` 等于 `t[j]` 的位置，此时 `i` 和 `j` 同时向右移动一位开启新一轮匹配
直到匹配完成或者 `j` 到达 `t` 的末尾


```cpp
class Solution {
public:
    bool isSubsequence(string s, string t) {
        int i = 0, j = 0;
        while(i < s.size() && j < t.size()){
            while(j < t.size() && s[i] != t[j]) j++;
            if(j < t.size() && s[i] == t[j]){
                i++;
                j++;
            }
        }
        return i == s.size();
    }
};
```


- 时间复杂度：${O\left( n+m \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
