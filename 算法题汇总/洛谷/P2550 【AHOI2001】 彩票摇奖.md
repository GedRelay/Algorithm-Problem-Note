# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, m, b, a[34] = { 0 }, p = 0, ans[8] = { 0 };
	cin >> n;
	for (int i = 1; i <= 7; i++)
	{
		cin >> m;
		a[m] = 1;
	}
	for (int i = 0; i < n; i++)
	{
		p = 0;
		for (int j = 1; j <= 7; j++)
		{
			cin >> b;
			if (a[b] == 1)
			{
				p++;
			}
		}
		ans[p]++;
	}
	for (int i = 7; i >= 1; i--)
	{
		cout << ans[i] << " ";
	}
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
