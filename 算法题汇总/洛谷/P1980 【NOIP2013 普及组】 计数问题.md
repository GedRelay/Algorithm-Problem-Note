# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, x, m[10] = { 0 };
	cin >> n >> x;
	for (int i = 1; i <= n; i++)
	{
		int j = i;
		while (j!=0)
		{
			m[j % 10]++;
			j = j / 10;
		}
	}
	cout << m[x];
	return 0;
}
```


- 时间复杂度：${O\left( n\log n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
