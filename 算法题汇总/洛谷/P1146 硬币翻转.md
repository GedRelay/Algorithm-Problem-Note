# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, a[101] = {0};
	cin >> n;
	cout << n << endl;
	for (int i = 1; i <= n; i++)
	{
		for (int j = 1; j <= n; j++)
			a[j] = !a[j];
		a[i] = !a[i];
		for (int j = 1; j <= n; j++)
			cout << a[j];
		cout << endl;
	}
	return 0;
}
```


- 时间复杂度：${O\left( n^{2}  \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
