# 思路
- 思路 1：
二分答案
函数：f (空旷指数) = 增设的路标数量
趋势：递减
条件：f (x) <= K
目标：找到第一个 x
使用模型 1


```cpp
#include<iostream>
#include<algorithm>
using namespace std;
typedef long long LL;
int L, N, K;
int a[100005];

int f(int x){
    // 空旷指数为x时，需要增设的路标数量
    int cnt = 0;
    for(int i = 1; i < N; i++){
        cnt += (a[i] - a[i - 1] - 1) / x;
    }
    return cnt;
}

int main(){
    cin >> L >> N >> K;
    for(int i = 0; i < N; i++){
        cin >> a[i];
    }
    int l = 1, r = L;
    while(l < r){
        int mid = l + r >> 1;
        if(f(mid) <= K) r = mid;
        else l = mid + 1;
    }
    cout << l << endl;
    return 0;
}

```


- 时间复杂度：${O\left( N\log L \right)  }$ 
- 空间复杂度：${O\left( N \right)  }$
