# 思路
- 思路 1：
如果一个数是 2 的幂，那么其二进制下只有一个 1
使用 `lowbit` 操作 `(n & -n)` 得到二进制下第一个 1，判断 n 是否与其相等即可


```cpp
class Solution {
public:
    bool isPowerOfTwo(int n) {
        return n > 0 && n == (n & -n);
    }
};
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
