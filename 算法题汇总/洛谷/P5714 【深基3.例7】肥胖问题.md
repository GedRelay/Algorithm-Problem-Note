# 思路
- 思路 1：


```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
	double m, h, BMI;
	cin >> m >> h;
	BMI = m / h / h;
	if (BMI < 18.5)cout << "Underweight";
	else if (BMI >= 24)cout << BMI << endl << "Overweight";
	else cout << "Normal";
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
