# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,ans=1;
	char a, b='0';
	cin >> n;
	for (int i = 0; i < n; i++)
	{
		cin >> a;
		if (a != b) 
			ans++;
		b = a;//b暂存a的字母
	}
	cout << ans;
	return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
