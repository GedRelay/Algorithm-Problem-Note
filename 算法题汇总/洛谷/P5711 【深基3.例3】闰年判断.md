# 思路
- 思路 1：


```cpp
#include<iostream>
using namespace std;
int main()
{
	int year;
	cin >> year;
	bool p = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
	cout << p;
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
