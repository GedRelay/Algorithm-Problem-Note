# 思路
- 思路 1：


```cpp
class Solution {
public:
    vector<int> runningSum(vector<int>& nums) {
        vector<int> presum(nums.size(), 0);
        for(int i = 0; i < nums.size(); i++){
            if(i == 0) presum[i] = nums[i];
            else presum[i] = presum[i - 1] + nums[i];
        }
        return presum;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
