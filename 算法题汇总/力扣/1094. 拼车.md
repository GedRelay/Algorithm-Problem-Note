# 思路
- 思路 1：

```cpp
class Solution {
public:
    bool carPooling(vector<vector<int>>& trips, int capacity) {
        vector<int> d(1001, 0);
        for(auto t : trips){
            d[t[1]] += t[0];
            d[t[2]] -= t[0];
        }
        for(int i = 0; i <= 1000; i++){
            if(i != 0) d[i] += d[i - 1];
            if(d[i] > capacity) return false;
        }
        return true;
    }
};
```


- 时间复杂度：${O\left( n+l \right)  }$, ${n }$ 为数组长度, ${l }$ 为时间最大范围
- 空间复杂度：${O\left( l \right)  }$
