# 思路
- 思路 1：
按照题目要求模拟即可


```cpp
#include<iostream>
using namespace std;
int main()
{
	int hand = 0, bank = 0, yusuan;
	for (int i = 1; i <= 12; i++){
		cin >> yusuan;
		hand += (300 - yusuan);
		if (hand < 0){
			cout << "-" << i;
			return 0;
		}
		while (hand >= 100){
			hand -= 100;
			bank += 100;
		}
	}
	cout << hand + bank * 1.2;
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
