# 思路
- 思路 1：
贪心
奇数的最后一位肯定是 `'1'`，要求最大，则其余 `'1'` 的位置要尽可能往前面放 
首先统计 `'1'` 的个数 `oneNum`，然后前 `oneNum-1` 个位置变为 `'1'`，其余位置填 `'0'`，最后一个位置填 `'1'` 


```cpp
class Solution {
public:
    string maximumOddBinaryNumber(string s) {
        int n = s.size(), oneNum = 0;
        for(char c : s)
            if(c == '1') oneNum++;
        for(int i = 0; i < n; i++)
            if(i < oneNum - 1) s[i] = '1';
            else s[i] = '0';
        s[n - 1] = '1';
        return s;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
