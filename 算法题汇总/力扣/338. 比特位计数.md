# 思路
- 思路 1：
动态规划
转移方程：
$$
dp\left[ i \right] =\left( i\&1 \right) +dp\left[ i> >1 \right] 
$$

```cpp
class Solution {
public:
    vector<int> countBits(int n) {
        vector<int> ans(n + 1);
        for(int i = 0; i <= n; i++){
            ans[i] = (i & 1) + ans[i >> 1];
        }
        return ans;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
