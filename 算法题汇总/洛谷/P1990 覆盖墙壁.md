# 思路
- 思路 1：
见题解第一篇


```cpp
#include<iostream>
using namespace std;

const int mod = 10000;
int n;
int dp1[1000010] = {1, 1};
int dp2[1000010] = {0, 1};

int main(){
    cin >> n;
    for(int i = 2; i <= n; i++){
        dp1[i] = (dp1[i - 1] + dp1[i - 2] + 2 * dp2[i - 2]) % mod;
        dp2[i] = (dp1[i - 1] + dp2[i - 1]) % mod;
    }
    cout << dp1[n];
    return 0;
}
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
