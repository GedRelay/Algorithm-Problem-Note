# 思路
- 思路 1：
模拟即可，使用哈希表判断某个数是否已经出现过，如果出现过则返回 `false` 

```cpp
class Solution {
public:
    bool isHappy(int n) {
        unordered_set<int> s;
        while(n != 1){
            s.insert(n);
            int sum = 0;
            while(n){
                sum += (n % 10) * (n % 10);
                n /= 10;
            }
            n = sum;
            if(s.count(n)) return false;
        }
        return true;
    }
};
```


- 时间复杂度：${O\left( \log n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
