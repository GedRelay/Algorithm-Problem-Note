# 思路
- 思路 1：
枚举每个四宫格，当存在四宫格中 `W` 或者 `B` 的数量为 `1` 或 `3` 时，则返回 `true`  


```cpp
class Solution {
public:
    bool canMakeSquare(vector<vector<char>>& grid) {
        for(int i = 0; i < 2; i++){
            for(int j = 0; j < 2; j++){
                int cnt = 0;
                for(int x = i; x <= i + 1; x++){
                    for(int y = j; y <= j + 1; y++){
                        if(grid[x][y] == 'B') cnt++;
                    }
                }
                if(cnt == 0 || cnt == 1 || cnt == 3 || cnt == 4) return true;
            }
        }
        return false;
    }
};
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
