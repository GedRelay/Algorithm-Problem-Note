# 思路
- 思路 1：
枚举所有 $10000\sim 30000$ 的五位数，逐个判断是否满足要求即可


```cpp
#include<iostream>
using namespace std;
int main(){
	int k;
	bool a = 0;
	cin >> k;
	for (int i = 10000; i <= 30000; i++){
		if (i / 100 % k == 0) {
			if ((i / 10 - i / 10000 * 1000) % k == 0) {
				if ((i - i / 1000 * 1000) % k == 0) { 
					cout << i << endl;
					a = 1; 
				}
			}
		}
	}
	if (a == 0) cout << "No";
	return 0;
}
```


- 时间复杂度：${O\left( 10^{4}  \right) =O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
