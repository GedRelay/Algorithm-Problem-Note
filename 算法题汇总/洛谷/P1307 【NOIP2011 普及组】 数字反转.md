# 思路
- 思路 1：


```cpp
#include <iostream>
using namespace std;
int main()
{
    int n,k=0;
    cin >> n;
    while(n)
    {
        k = k * 10 + n % 10;
        n /= 10;
    }
    cout << k << endl;
    return 0;
}
```


- 时间复杂度：${O\left( \log n \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
