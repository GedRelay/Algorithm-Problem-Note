# 思路
- 思路 1：
搜索 ${9 }$ 个数字的所有可能的排列，然后分别判断是否满足比例条件
可以使用 `next_permutation` 遍历所有排列


```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long LL;
int A, B, C, cnt;
int a[10];
int main()
{
	cin >> A >> B >> C;
	for (int i = 1; i <= 9; i++)
		a[i] = i;
	do
	{
		LL x = a[1] * 100 + a[2] * 10 + a[3];
		LL y = a[4] * 100 + a[5] * 10 + a[6];
		LL z = a[7] * 100 + a[8] * 10 + a[9];
		if (x * B == y * A && x * C == z * A)
			cout << x << " " << y << " " << z << endl, cnt++;
	} while (next_permutation(a + 1, a + 10));
	if (!cnt)cout << "No!!!";
	return 0;
}
```
