# 思路
- 思路 1：
按照题目模拟即可


```cpp
#include<iostream>
#include<cmath>
using namespace std;
int main(){
	double n;
	cin >> n;
	n = ceil(log(1 - 0.01*n) / log(0.98));
	cout << n;
	return 0;
}
```


- 时间复杂度：${O\left( 1 \right)  }$ 
- 空间复杂度：${O\left( 1 \right)  }$
