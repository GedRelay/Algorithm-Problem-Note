# 思路
- 思路 1：
哈希表存储是宝石的字符，再遍历 `stones` 判断即可


```cpp
class Solution {
public:
    int numJewelsInStones(string jewels, string stones) {
        int cnt = 0;
        unordered_map<char,int> mp;
        for(char c:stones) mp[c]++;
        for(char c:jewels) cnt += mp[c];
        return cnt;
    }
};
```


- 时间复杂度：${O\left( n \right)  }$ 
- 空间复杂度：${O\left( n \right)  }$
