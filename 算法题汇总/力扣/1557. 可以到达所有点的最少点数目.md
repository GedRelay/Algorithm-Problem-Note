# 思路
- 思路 1：
找到所有入度为 $0$ 的点，将其加入答案 

```cpp
class Solution {
public:
    vector<int> findSmallestSetOfVertices(int n, vector<vector<int>>& edges) {
        vector<int> in(n);
        for(auto e : edges)
            in[e[1]]++;
        vector<int> ans;
        for(int i = 0; i < n; i++)
            if(in[i] == 0)
                ans.push_back(i);
        return ans;
    }
};
```


- 时间复杂度：${O\left( n+m \right)  }$ , ${n }$ 为点数, ${m }$ 为边数
- 空间复杂度：${O\left( n \right)  }$
